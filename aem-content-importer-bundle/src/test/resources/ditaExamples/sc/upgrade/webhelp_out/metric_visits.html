Updated Visits Metric for Typed Bookmarked in Referring Domains Report

If you had the Visits metric enabled on the Referring Domains report in SiteCatalyst 14, you’ll notice a change to the Visit count on this report after upgrading to SiteCatalyst 15.

In SiteCatalyst 14, Typed/Bookmarked was counted anytime a visit contained a server call with no referrer specified, regardless of where the server call occurred during the visit.

Since the referrer can often be empty mid-visit due to actions such as Flash navigation and redirects, SiteCatalyst 15 was improved to count Typed/Bookmarked only when a visit starts with a server call with no referrer specified.

With this change, in SiteCatalyst 15 Instances of Typed/Bookmarked should match Visits, whereas, in SiteCatalyst 14, Instances were typically lower than Visits.
