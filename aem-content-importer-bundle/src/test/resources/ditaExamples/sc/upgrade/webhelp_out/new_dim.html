Add New Dimensions and Metrics

You can add new dimensions and metrics from Test&Target, Survey, and list variables.

A second visit metric called Visit – All Visitors is available in DataWarehouse. SiteCatalyst 15 defines visits as all cookie and non-cookie visitors. See Visits Metric Includes Non-cookie Visitors. DataWarehouse is compatible with SiteCatalyst 14 and SiteCatalyst 15, so the Visits metric is preserved. If you want to view all cookie and non-cookie visitors in a request, select Visits – All Visitors.
