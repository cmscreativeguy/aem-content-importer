Managing Segments Prior to Upgrading

SiteCatalyst 15 provides custom segments and global access to shared Discover segments.

Before upgrading to SiteCatalyst 15, Adobe recommends auditing all segments that are shared by Discover users, and selecting which segments should appear in the SiteCatalyst segments list. Any segments that should not be shared with all SiteCatalyst users should be moved to a non-shared folder. Shared segments might need to be renamed since they are visible to all SiteCatalyst users. For example, you might prefix global segments with CORP: or GLOBAL:, and personal segments with your name.

Note: You cannot apply segments to a legacy dashboard.
