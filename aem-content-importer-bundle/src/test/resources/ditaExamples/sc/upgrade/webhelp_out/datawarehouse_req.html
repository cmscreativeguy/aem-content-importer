DataWarehouse Requests

After you upgrade to SiteCatalyst 15, VISTA and processing rules are applied at data collection time, rather than at report run-time.

DataWarehouse data reflects native SiteCatalyst VISTA processing, and reports with VISTA rules applied run considerably faster. This improvement can be enabled for SiteCatalyst14 as well.

  • Create Duplicate Requests
    You can create requests from the Request Manager tab.
  • Prioritize Requests
    You can prioritize requests using a drag-and-drop interface.
  • View Detailed Status
    You can click the report name to view a detailed status.
  • Add New Dimensions and Metrics
    You can add new dimensions and metrics from Test&Target, Survey, and list variables.
