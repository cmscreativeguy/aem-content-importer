Post-Upgrade Checklist

The following list contains tasks to verify your SiteCatalyst 15.

These should be performed after your scheduled upgrade date.

  1. Run a report that spans the upgrade date and look for a message that indicates that some of the data was processed by a previous platform. This is the message that appears in SiteCatalyst 15 when you view data that was collected by SiteCatalyst 14.
  2. Verify that segments appear in the drop-down. If you see unexpected segments or if you would like to change the list of segments that appear, review Managing Segments.
  3. Check the Visits metric to ensure the increases is similar to what was expected.
  4. Instruct users to select Marketing Cloud/SiteCatalyst 15 on the login page. The default version in the drop-down does not change automatically.
