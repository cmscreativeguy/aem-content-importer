Feature Overview

SiteCatalyst 15 provides several workflow and performance improvements.

You can apply real-time visitor segmentation, integrate more data, and see complex reports return more quickly.

  • Site Overview Report
    The new Suite home page is a dynamically generated report called the Site Overview report.
  • Suite Product Access
    Accessing products and features has changed. To access Marketing Cloud products, click Adobe Marketing Cloud, located above the left navigation.
  • Segmentation
    Segmentation lets you define custom data subsets and generate reports using that data subset.
  • Full Subrelations
    Full subrelations are enabled on all conversion reports, so you can break down any eVar by another eVar.
  • Default Metrics on Reports
    Administrators can apply default metrics at the report level.
  • Metric Changes
    Metric changes between SiteCatalyst 14 and SiteCatalyst 15.
  • Report Menu Search
    You can quickly locate reports, metrics, and dashboards, using the menu's Search field in the left navigation.
  • Video Report Changes
    Standard video reports have been replaced with new overview and detail reports.
  • DataWarehouse Requests
    After you upgrade to SiteCatalyst 15, VISTA and processing rules are applied at data collection time, rather than at report run-time.
  • Processing Rules
    Certified administrators can change SiteCatalyst data as it is received, which simplifies interaction with IT groups and Web developers.
  • Context Data Variables
    Context Data variables let you define variables that can be read by processing rules.
  • List Variables
    List variables are a delimited list of values that are passed into a variable, then reported as individual line items for reporting.
  • User Management Enhancements
    Transfer user privileges, log in as a different user, and see who is logged in to Discover.
  • Bot Traffic Filtering
    Traffic from known bots and spiders can now be removed from report suites.
  • Data Visibility in Other SiteCatalyst Versions
    After you migrate to SiteCatalyst 15, you can continue to view new and historical data in other versions of SiteCatalyst.
